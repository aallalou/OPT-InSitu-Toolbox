This directory contains pre-built Python modules for the ASTRA Toolbox.

It has been built with WinPython-64bit-3.4.2.

To use it, move the astra/ directory to your existing site-packages directory.
Its exact location depends on your Python installation, but should look
similar to C:\WinPython\python-3.4.2.amd64\Lib\site-packages .

Sample code can be found in the examples/ directory.



The web page for the Python interface to the ASTRA Toolbox can be found at

http://dmpelt.github.io/pyastratoolbox
